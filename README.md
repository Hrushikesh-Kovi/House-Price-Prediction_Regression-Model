# 🏠 House Price Prediction

A Machine Learning project focused on predicting house prices using regression models.  
The project involves data cleaning, exploratory data analysis, preprocessing, and model training to understand how different features influence housing prices.

## ⚙️ Work Done

The following steps were performed in this project:

- 🧹 **Data Cleaning**
  - Handling missing values
  - Removing unnecessary columns

- 🔄 **Data Preprocessing**
  - Encoding categorical variables
  - Preparing data for machine learning models

- 📏 **Feature Scaling**
  - Standardization/Normalization of features

- 🔍 **Exploratory Data Analysis (EDA)**
  - Data visualization
  - Understanding feature relationships
  - Identifying patterns in housing prices

- 🤖 **Model Training**
  - Training regression models on the dataset

- 📊 **Model Evaluation**
  - Evaluating model performance using suitable metrics
  - Comparing results to understand model effectiveness

---

## 📈 Key Insights

Some observations discovered during analysis:

- Certain house features have a strong correlation with price.
- Proper feature preprocessing improves model performance.
- Data visualization helps in understanding patterns and outliers in housing data.
